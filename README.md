### 外貌特征
战斗形态下，他身披银蓝相间的金属铠甲，铠甲线条刚硬且带有尖刺状设计，充满攻击性。头部有尖锐突起，眼部闪烁红光，给人冷酷凶狠的感觉。
<br/>
非战斗形态时，他皮肤呈深蓝色，黑色的头发根根竖起，眼神锐利，肩部铠甲宽大且造型独特，胸口有红色菱形装饰，整体造型霸气威严 。

+ 特征1：皮肤呈深蓝色
+ 特征2：眼神凶狠
